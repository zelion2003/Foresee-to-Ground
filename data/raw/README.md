# Raw External Inputs

This directory is a placeholder for raw files that are not distributed with the repository.

Recommended layout:

- `timelens/timelens-100k.jsonl`
- `timelens_bench/activitynet-timelens.json`
- `timelens_bench/charades-timelens.json`
- `timelens_videos/`

The conversion tools under `tools/` use these locations as their default input paths.
