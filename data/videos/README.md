# Video Placement

Video files are not distributed with this repository.

Please download or prepare the datasets separately and place them in the following directories:

- `anet/`
- `didemo/`
- `internvid/`
- `timelens_100k_336/`
- `timelens_bench_336/`

The training scripts assume these locations unless you explicitly override them with command-line arguments or environment variables.
